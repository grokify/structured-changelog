# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html),
commits follow [Conventional Commits](https://www.conventionalcommits.org/en/v1.0.0/),
and this changelog is generated by [Structured Changelog](https://github.com/grokify/structured-changelog).

## [Unreleased]

## [v0.8.0] - 2026-01-18

### Highlights

- Monorepo tag path support for repositories using path-prefixed version tags (e.g., `sdk/go/v1.0.0`)

### Added

- `tag_path` field in changelog IR for specifying version tag prefixes in monorepos
- Automatic path prefixing for compare and tag links when `tag_path` is set
- `tag_path` property in JSON schema with documentation

### Tests

- Comprehensive tests for tag path functionality including GitHub, GitLab, unreleased links, and edge cases

## [v0.7.0] - 2026-01-11

### Highlights

- CLI renamed from `sclog` to `schangelog` for naming consistency with sibling tools (sroadmap, sprd, strd)

### Changed

- **BREAKING:** CLI binary renamed from `sclog` to `schangelog` for consistency with sibling structured document tools ([`e58d452`](https://github.com/grokify/structured-changelog/commit/e58d452))

### Documentation

- Updated all documentation, examples, and help text to use `schangelog` ([`e58d452`](https://github.com/grokify/structured-changelog/commit/e58d452))
- Added git log alternatives section to project CLAUDE.md recommending `schangelog` over raw git commands ([`e58d452`](https://github.com/grokify/structured-changelog/commit/e58d452))

## [v0.6.0] - 2026-01-10

### Highlights

- TOON output format for LLM commands (~8x token reduction vs raw git log)
- GitLab repository support with automatic reference linking for issues, PRs, and commits
- Author attribution for external contributors and compact grouping of maintenance releases

### Added

- TOON (Token-Oriented Object Notation) as default output format for `parse-commits`, `suggest-category`, and `validate --format` ([`2be0cc0`](https://github.com/grokify/structured-changelog/commit/2be0cc0))
- `--format` flag for LLM commands supporting `toon` (default), `json`, and `json-compact` ([`2be0cc0`](https://github.com/grokify/structured-changelog/commit/2be0cc0))
- GitLab repository support for reference links and version comparisons ([`be9b3b0`](https://github.com/grokify/structured-changelog/commit/be9b3b0))
- Automatic reference linking: issues, PRs, and commits become clickable links when repository URL is provided ([`be9b3b0`](https://github.com/grokify/structured-changelog/commit/be9b3b0))
- `LinkReferences` renderer option (now enabled by default) for hyperlinking references ([`be9b3b0`](https://github.com/grokify/structured-changelog/commit/be9b3b0))
- `IncludeUnreleasedLink` renderer option (enabled by default) for always showing compare link to HEAD
- `versioning` field in changelog IR: `semver`, `calver`, `custom`, or `none` ([`93b8da1`](https://github.com/grokify/structured-changelog/commit/93b8da1))
- `commit_convention` field in changelog IR: `conventional` or `none` ([`93b8da1`](https://github.com/grokify/structured-changelog/commit/93b8da1))
- Dynamic header generation based on versioning scheme and commit convention ([`be9b3b0`](https://github.com/grokify/structured-changelog/commit/be9b3b0))
- Validation error codes E008 (invalid versioning) and E009 (invalid commit convention) ([`93b8da1`](https://github.com/grokify/structured-changelog/commit/93b8da1))
- `maintainers` and `bots` fields in changelog IR for identifying team members ([`91e788f`](https://github.com/grokify/structured-changelog/commit/91e788f))
- Auto-detection of common bots (dependabot, renovate, github-actions, etc.) ([`91e788f`](https://github.com/grokify/structured-changelog/commit/91e788f))
- `IsTeamMember()` method for checking if author is maintainer or bot ([`91e788f`](https://github.com/grokify/structured-changelog/commit/91e788f))
- `IncludeAuthors` renderer option for external contributor attribution ([`a2c86cb`](https://github.com/grokify/structured-changelog/commit/a2c86cb))
- Automatic author attribution with linked GitHub/GitLab profiles ([`a2c86cb`](https://github.com/grokify/structured-changelog/commit/a2c86cb))
- Inline attribution stripping to avoid duplicate attribution when author field matches ([`b5e8f34`](https://github.com/grokify/structured-changelog/commit/b5e8f34))
- `CompactMaintenanceReleases` renderer option for grouping maintenance-only releases ([`f7cdb02`](https://github.com/grokify/structured-changelog/commit/f7cdb02))
- `IsMaintenanceOnly()` method to detect dependency/documentation-only releases ([`824a075`](https://github.com/grokify/structured-changelog/commit/824a075))
- `--changelog` flag for `parse-commits` to auto-detect external contributors ([`817fa7f`](https://github.com/grokify/structured-changelog/commit/817fa7f))
- `IsExternal` field in commit output marking external contributors ([`8e1ab0a`](https://github.com/grokify/structured-changelog/commit/8e1ab0a))
- `Contributors` summary in `parse-commits` output with deduplicated author list ([`8e1ab0a`](https://github.com/grokify/structured-changelog/commit/8e1ab0a))
- `IsTeamMemberByNameAndEmail()` for matching by git author name, email, or GitHub noreply username ([`5169681`](https://github.com/grokify/structured-changelog/commit/5169681))

### Changed

- **BREAKING:** `parse-commits` and `suggest-category`: `--compact` flag replaced with `--format` flag ([`2be0cc0`](https://github.com/grokify/structured-changelog/commit/2be0cc0))

- Consecutive maintenance releases grouped into compact sections by default ([`f7cdb02`](https://github.com/grokify/structured-changelog/commit/f7cdb02))
- Version strings now used as-is in headers and links (no automatic `v` prefix)

- CLI flags: `--max-tier` for generate, `--min-tier` for validate ([`b185758`](https://github.com/grokify/structured-changelog/commit/b185758))
- Library functions: `Summary()`, `ValidateMinTier()`, `ParseTier()`, `OptionsFromConfig()` ([`b185758`](https://github.com/grokify/structured-changelog/commit/b185758))
